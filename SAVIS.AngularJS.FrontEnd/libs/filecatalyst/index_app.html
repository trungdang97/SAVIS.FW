TransferAgent

This sample application provides a web framework which in turn hosts example instances of FileCatalyst TransferAgent. The first time you visit such a page, you will be presented with some basic configuration questions and dialogs, and the opportunity to download the FileCatalyst TransferAgent application. After that, accelerated file transfers are available right from within the web browser.
